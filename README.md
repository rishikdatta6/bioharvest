# BioHarvest 🌱

A full-stack crop analytics platform that analyzes soil, AQI, and weather data to provide smart harvest decisions.

## Tech Stack

- Frontend: React (Vercel)
- Backend: ASP.NET Core Web API (Azure)
- Database: Azure SQL
- Auth: JWT + Identity

## Features

- User Registration & Login
- JWT Authentication
- Forgot & Reset Password via Email
- Crop Analysis (Soil, AQI, Weather)
- Secure API integration

## Live Demo

🌐 App: https://bioharvest-phi.vercel.app

## API
Backend is hosted on Azure App Service and powers authentication, data processing, and analytics.
